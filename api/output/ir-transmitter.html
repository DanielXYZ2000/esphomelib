IR Transmitter¶

Example Usage¶

// Create the hub
auto *ir = App.make_ir_transmitter_component(32);
// Create switches
auto *panasonic_on = ir->create_transmitter(SendData::from_panasonic(0x4004, 0x100BCBD).repeat(25));
App.make_mqtt_switch_for("Panasonic TV On", panasonic_on);
App.make_mqtt_switch_for("Panasonic TV Volume Up", ir->create_transmitter(SendData::from_panasonic(0x4004, 0x1000405)));

See Application::make_ir_transmitter_component() and Application::make_mqtt_switch_for().

class esphomelib::output::IRTransmitterComponent¶

This is an IR Transmitter hub using the ESP32 RMT peripheral.

With it you can send out IR commands from different vendors and have them appear as switches in Home Assistant.
