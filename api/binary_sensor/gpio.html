GPIO Binary Sensor¶

Example Usage¶

// Basic
App.make_gpio_binary_sensor(36, "Window Open",
                            binary_sensor::device_class::WINDOW);
// Custom pinMode
App.make_gpio_binary_sensor(GPIOInputPin(36, INPUT_PULLUP), "Window Open",
                            binary_sensor::device_class::WINDOW);

See Application::make_gpio_binary_sensor().
