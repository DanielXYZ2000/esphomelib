API Reference¶

Switch¶

class esphomelib::switch_::Switch¶

Base class for all switches.

A switch is basically just a combination of a binary sensor (for reporting switch values) and a write_state method that writes a state to the hardware.

Inherits from esphomelib::binary_sensor::BinarySensor

Subclassed by esphomelib::switch_::IRTransmitterComponent::DataTransmitter, esphomelib::switch_::RestartSwitch, esphomelib::switch_::SimpleSwitch
